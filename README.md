🌳 线程
程序:为完成特定的任务，用某种语言编写的一组指令的集合
进程:是程序的依次执行或是正在运行的一个程序
线程:进程可以京一部细化为线程，是一个程序内部的一条执行路径

重点:
😊线程的创建和使用
😊线程的生命周期
😊线程的同步
😊线程的通信

重点:
一个Java应用程序java.exe，其实至少有三个线程，main()主线程,gc()垃圾回收线程，异常处理线程。如果发生异常会影响main主线程

重点:
并行:多个CPU同时执行多个任务
并发:一个CPU（采用时间片）同时执行多个任务
